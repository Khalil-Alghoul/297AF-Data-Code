# 297AF-Data-Code
Cytoscape Network.jpeg
Protein similarity network visualization generated in Cytoscape, showing relationships and clustering among AmpC β-lactamase sequences.

KEGG results.jpeg
Screenshot of KEGG pathway annotation confirming all sequences are associated with β-lactam resistance (KO: K01467).

Phylogenetic Analysis tree.jpeg
Phylogenetic tree constructed using the Neighbor-Joining method with bootstrap analysis, illustrating evolutionary relationships among sequences.

README.md
Project overview, including methodology, dataset description, and instructions for reproducing the analysis.

ampC_filtered.fasta
Final curated dataset of 25 AmpC β-lactamase protein sequences used for alignment and downstream analysis.

ampc.pdb
3D protein structure of AmpC β-lactamase used for structural analysis and inhibitor binding evaluation.

blast_raw_sequences.fasta
Initial set of sequences retrieved from BLAST prior to filtering and cleaning.

clustalo-120260330-143645-0
Multiple sequence alignment output generated by Clustal Omega, showing conserved and variable regions across sequences.

filter_sequences.py
Python script used to filter BLAST results based on identity thresholds and sequence quality criteria.

iprscan5-R20260412-001644-
InterProScan output file containing domain and functional annotations for the analyzed sequences.

ligand_avibactam.pdb
3D structure of avibactam used for docking and inhibitor binding analysis.

ligand_clavulanic.pdb
3D structure of clavulanic acid used for docking and inhibitor binding analysis.
